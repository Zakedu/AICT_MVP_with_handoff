## 1. 서비스 개요

### 1.1 서비스 정의
**AICT Essential**은 직장인의 AI 활용 역량을 공인 검증하는 온라인 인증 시험 플랫폼입니다.

### 1.2 서비스 비전
> "AI 시대, 당신의 역량을 증명하세요"

- **객관적 역량 검증**: 6가지 AI 역량 지표 기반 표준화된 평가
- **직무 맞춤형 평가**: 10개 직무별 실무 시나리오로 현장 적합성 측정
- **공인 디지털 인증**: LinkedIn 연동, QR 검증 가능한 디지털 배지 발급
- **부정행위 방지**: 화면 녹화 + 탭 이탈 감지 + 사후 검수 시스템

### 1.3 핵심 지표 (6가지 역량)

| 역량 | 영문명 | 설명 | 측정 파트 |
|------|--------|------|----------|
| AI 개념 이해 | Defining | AI 기술과 용어에 대한 기본 이해 | Part 1 |
| 프롬프트 설계 | Prompting | 효과적인 AI 지시문 작성 능력 | Part 1, 2, 3 |
| 데이터 보호 | Protecting | 개인정보 및 기밀정보 보호 의식 | Part 1, 3 |
| 결과 검증 | Refining | AI 출력물 검토 및 개선 능력 | Part 2, 3 |
| 윤리적 판단 | Acumen | 저작권, 책임, 편향 인식 | Part 1, 3 |
| 업무 통합 | Integrating | 실무 프로세스에 AI 적용 능력 | Part 2, 3 |

### 1.4 인증서 정책

| 항목 | 내용 |
|------|------|
| 합격 기준 | 100점 만점 중 70점 이상 |
| 유효 기간 | 발급일로부터 1년 |
| 재응시 | 불합격 시 7일 후 재응시 가능 |
| 인증 형태 | 디지털 배지 + PDF 인증서 |
| 검증 방법 | QR코드 스캔 → 검증 페이지 |
